## 🚀 How to build your first app

Once the app opens, you will see a text field. This is the prompt bar. Type your app idea here. Be as specific as you can. For example, you might type "Build me a task list app with categories for home and work." 

The software processes your request. It generates the app interface and the internal logic. You can see a preview of your app in the center of the screen as it builds. If you want to change the color, the layout, or the features, type a new instruction in the bar. The software updates the app based on your latest prompt.

## 📁 Managing your project files

The software saves your progress automatically. Each time you create a new app, it stores the content in a folder within your Documents directory. You can find these files by clicking the folder icon in the top menu of the dashboard. Do not move or rename these files while the application is open. This prevents data loss.

## 🛠️ Troubleshooting common issues

If the app fails to start, verify your internet connection. The application requires a active connection to communicate with the model servers. If the preview window stays blank, try restarting the application. Close the app completely and open it again after ten seconds. 

If you receive a notification about missing permissions, ensure that your user account has administrative rights. Windows may ask for permission during installation. Click Yes to allow the installer to place the necessary files on your hard drive. 

Check for temporary files if the app runs slowly. Over time, the software builds cache data. You can clear this cache in the Options panel. This process does not delete your projects. It only removes temporary data files.

## 📡 Updates and version control

The software checks for updates when you launch it. If a new version exists, you will see a prompt. Click Update to apply the latest changes. This process does not remove existing projects. It replaces the core files with the latest versions to improve stability. Always keep your software updated to ensure compatibility with the cloud servers.

## 💬 Frequently Asked Questions

Can I build apps for other platforms? 
This software focuses on iOS apps. You can view the layout on your desktop but the final package remains optimized for iPhone deployment.

What happens if I lose my internet connection?
The software pauses the build process. It resumes once you reconnect to the network. Your progress stays saved in the local folder.

Is my data private?
Your project prompts and app files remain on your local machine. Only the necessary snippets sent to the intelligence model get encrypted during transmission.

Do I need to pay for the intelligence service?
Access to the model is included in this software package. You do not need to sign up for external subscriptions.

Can I move my app files to other computers?
You can copy the project folder to a new computer. Install the app on the new machine and open the project from the folder. All features will load in the correct state.
